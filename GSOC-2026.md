# GSoC 2026 Project Journal: RoaringBitmap Functions for Apache Fluss

This file serves as the central hub for my Google Summer of Code 2026 project. Instead of a static document, this is a living journal to track my progress, challenges, and learnings as I implement native RoaringBitmap functions in Apache Fluss.

## My Mission

My goal is to make powerful bitmap-based analytics a seamless, out-of-the-box experience for Apache Fluss users. By implementing and registering a core set of RoaringBitmap functions directly into the Flink SQL layer, I will remove the need for custom UDFs and reduce the friction of performing high-performance, real-time analytics.

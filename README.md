# GSoC 2026: Native RoaringBitmap Integration for Apache Fluss

Welcome to my Google Summer of Code 2026 project repository!

## About Me

I am **Prajwal Banakar**, a recent graduate with a degree in Information Science & Engineering (B.E.). I have a strong interest in distributed systems, real-time data infrastructure, and backend engineering. I am a contributor to Apache Fluss.

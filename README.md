# GSoC 2026: Native RoaringBitmap Integration for Apache Fluss

Welcome to my Google Summer of Code 2026 project repository!

## About Me

I am **Prajwal Banakar**, a recent graduate with a degree in Information Science & Engineering (B.E.). I have a strong interest in distributed systems, real-time data infrastructure, and backend engineering. I am a contributor to Apache Fluss.

## About My Project

This repository documents my work for GSoC 2026 with Apache Fluss. My project is titled "Native RoaringBitmap Integration for Apache Fluss".

You can find a detailed journal of my project, including my progress, challenges, and learnings, in the [GSoC 2026 Project Journal](./GSOC-2026.md).

For details on my preparation and contributions leading up to this project, please see my [GSOC Preparation Document](./GSOC-Preperation.md).
